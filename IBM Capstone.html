Data Analyst Portfolio by Mayur Dusane

IBM Capstone Project with Python and IBM Cognos Analytics

Introduction

Created a dashboard in IBM Cognos Analytics to identify present and future trends in programming languages and platforms using job data. Collected job data from the data source using Rest API. Cleaned and performed exploratory data analysis on collected data using python language (Pandas, NumPy, Seaborn) in Jupyter Notebook.

Executive Summary

➢ The project involved gathering job demand data across different technologies and locations. Through rigorous analysis, valuable insights and trends were identified. This encompassed determining the top programming languages, in-demand database skills, and popular integrated development environments (IDEs).
➢ The findings were consolidated into a comprehensive dashboard that showcased current skills, future trending skills, and demographic information, providing a holistic view of the job market landscape.

Problem Statement

The data analyst portfolio project aims to address the challenge of understanding job market demand across different technologies and locations. By collecting and analyzing job opening data, the objective is to identify insights and trends, particularly focusing on the top programming languages, databases skills, and popular Integrated Development Environments (IDEs). The project seeks to consolidate this information into a comprehensive dashboard that showcases the current skills in demand, future trend skills, and demographic information, providing valuable insights for career planning and skill development in the technology industry.
